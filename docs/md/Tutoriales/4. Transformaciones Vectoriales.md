# 4.1 Vectores
Los vectores son direcciones y tienen una magnitud. Su magnitud también se llama modulo. \n
![Escritura de un Vector](vector.png)

## Producto escalar (dot product)
Sirve para saber el ángulo entre dos vectores, si es 0 significa que son perpendiculares y 1 si tienen el mismo ángulo, es decir paralelos o coincidentes.
![Fórmula Producto Escalar](porductoEsc.png)
Luego para calcular el ángulo basta con hacer arcsen(valor)
## Producto vectorial
Se define para 3 dimensiones, necesita 2 vectores no paralelos y produce un tercer vector ortogonal a los anteriores
