\begin{DoxyItemize}

\item {\bfseries{Offset (Desplazamiento inicial):\+}} 0 bytes 
\end{DoxyItemize}\hypertarget{pasos_render_autotoc_md15}{}\doxysubsubsection{\texorpdfstring{Detalles del formato de datos}{Detalles del formato de datos}}\label{pasos_render_autotoc_md15}

\begin{DoxyItemize}
\item Los datos de posición se almacenan como valores de punto flotante ({\itshape float}) de 32 bits (4 bytes).
\item Cada posición está compuesta por 3 de esos valores (X, Y, Z).
\item No hay espacio (ni otros valores) entre cada conjunto de 3 valores. Los valores están empaquetados de forma compacta ({\itshape tightly packed}) en el arreglo.
\item El primer valor de los datos se encuentra exactamente al principio del búfer.
\end{DoxyItemize}
