This skeleton uses the following libraries:\+


\begin{DoxyItemize}
\item \href{https://www.glfw.org/}{\texttt{glfw}} for managing Open\+GL windows
\item \href{https://glm.g-truc.net/}{\texttt{glm}} for the CPU mathematical operations that follow the specifications of the GPU mathematical operations in Open\+GL
\item \href{https://glew.sourceforge.net/}{\texttt{glew}} for loading Open\+GL extensions
\item \href{https://github.com/nothings/stb}{\texttt{stb\+\_\+image}} for loading and saving images
\end{DoxyItemize}

When creating a new project, remember to add the property sheet {\ttfamily IGProjects.\+props} to it. In order to add a new library, copy its directory into the solution root and update the elements {\ttfamily Include\+Path}, {\ttfamily Library\+Path}, {\ttfamily Additional\+Dependencies}, and {\ttfamily Local\+Debugger\+Environment} in {\ttfamily IGProjects.\+props}.

You can also create new property sheets.
